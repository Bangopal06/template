# Cara Install Aplikasi di Desktop & HP

Aplikasi ini sudah mendukung PWA (Progressive Web App) dan bisa diinstall seperti aplikasi native!

## 📱 Install di HP (Android/iOS)

### Android (Chrome):
1. Buka aplikasi di browser Chrome
2. Klik menu (3 titik) di pojok kanan atas
3. Pilih **"Tambahkan ke layar utama"** atau **"Install app"**
4. Klik **"Install"**
5. Aplikasi akan muncul di home screen seperti aplikasi biasa

### iOS (Safari):
1. Buka aplikasi di browser Safari
2. Tap tombol **Share** (kotak dengan panah ke atas)
3. Scroll dan pilih **"Add to Home Screen"**
4. Tap **"Add"**
5. Aplikasi akan muncul di home screen

## 💻 Install di Desktop

### Windows/Mac/Linux (Chrome/Edge):
1. Buka aplikasi di browser Chrome atau Edge
2. Lihat icon **install** (⊕) di address bar (sebelah kanan URL)
3. Klik icon tersebut
4. Klik **"Install"**
5. Aplikasi akan terbuka di window terpisah seperti aplikasi desktop

### Atau melalui menu:
1. Klik menu (3 titik) di pojok kanan atas
2. Pilih **"Install Dashboard Template"** atau **"Apps" → "Install this site as an app"**
3. Klik **"Install"**

## ✨ Keuntungan Install sebagai App:

- ✓ Akses cepat dari home screen/desktop
- ✓ Berjalan di window terpisah (tanpa address bar browser)
- ✓ Notifikasi push (jika diaktifkan)
- ✓ Bekerja offline (cache otomatis)
- ✓ Lebih cepat karena cache
- ✓ Terasa seperti aplikasi native

## 🔄 Update Aplikasi:

Aplikasi akan otomatis update saat Anda buka dan ada versi baru tersedia.

## 🗑️ Uninstall:

### Android:
- Tekan lama icon aplikasi → Pilih "Uninstall" atau "Hapus"

### iOS:
- Tekan lama icon aplikasi → Pilih "Remove App"

### Desktop:
- Buka aplikasi → Menu (3 titik) → "Uninstall Dashboard Template"
- Atau dari Settings browser → Apps → Uninstall

**Note:** Pastikan aplikasi sudah di-deploy ke server (Vercel/Netlify/dll) agar bisa diakses dari mana saja!
